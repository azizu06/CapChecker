# Triage labels

| Matt Pocock role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Needs maintainer evaluation |
| `needs-info` | `needs-info` | Waiting for missing information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and safe for an agent to pick up |
| `ready-for-human` | `ready-for-human` | Needs human implementation or judgment |
| `wontfix` | `wontfix` | Will not be actioned |

Additional execution labels are `lane-a-pipeline`, `lane-b-ui`, `lane-shared`,
`type-afk`, `type-hitl`, `priority-core`, and `priority-stretch`.
